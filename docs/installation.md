# Installation

## Stable release

To install Care Communication, run this command in your terminal:

```sh
pip install Care Communication
```

## From source

The source files for Care Communication can be downloaded from the [Github repo](https://github.com/ohcnetwork/care_communication).

You can either clone the public repository:

```sh
git clone git://github.com/ohcnetwork/care_communication
```

Or download the [tarball](https://github.com/ohcnetwork/care_communication/tarball/master):

```sh
curl -OJL https://github.com/ohcnetwork/care_communication/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd care_communication
pip install .
```
